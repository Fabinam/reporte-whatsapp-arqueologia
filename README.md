# Reporte WhatsApp Arqueología

App Streamlit para convertir reportes de WhatsApp en un Excel con:

- Para copiar
- Detalle normalizado
- Alertas

## Archivos

- app.py
- requirements.txt

## Despliegue

Usar Streamlit Community Cloud y seleccionar:

- Repository: reporte-whatsapp-arqueologia
- Branch: main
- Main file path: app.py
